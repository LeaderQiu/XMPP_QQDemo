# SXTheQQ
用xmppFramework框架编写QQ程序，主要为了练习通讯的一些原理，界面比较渣
必须要先在本地配置好环境才可以运行。配置方法见我博客园：<br />

`http://www.cnblogs.com/dsxniubility/p/4304570.html`


####主要功能
**1.用户注册**<br />
**2.用户上线下线检测**<br />
**3.用户加好友与被加好友**<br />
**4.发送文本信息**<br />
**5.发送图片和声音信息**<br />
